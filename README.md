# Supply Chain Demand Prediction (Supervised ML)

##  Problem Statement
Accurately predicting daily demand is critical for inventory planning, logistics optimization, and cost reduction in supply chains.  
This project formulates demand prediction as a **supervised classification problem** by discretizing continuous demand values into demand categories.

##  Dataset
- Supply chain operational dataset
- Target variable: **Daily Demand**
- Features include logistics, inventory, and operational indicators
